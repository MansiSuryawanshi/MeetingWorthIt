## 💡 The Problem

We've all been there.

An 8-person meeting. 1 hour. Everyone nodding. Nothing decided.

**What nobody thinks about?** That meeting just cost your company **$464.** Every week? **$24,125 a year.** That's a full-time intern — sitting in a conference room.

MeetingWorthIt makes that cost impossible to ignore.

---

## ✨ What It Does

### 📅 Meeting Cost Calculator
Input your meeting agenda, attendees, and their salaries — and watch the real dollar cost appear **in real time** as you add people. The number climbing as you add attendees? That's the point.

### 🤖 AI Attendance Analyzer
Powered by **Claude API**, the app breaks down:
- ✅ Who **must** be in the room
- 📩 Who can just **receive notes**
- 💬 Whether the whole thing could've been a **Slack thread**

Role by role. Reason by reason. No fluff.

### 📆 Calendar Import
Connect your calendar and instantly audit every **recurring meeting** draining your team's budget — before it drains another quarter.

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **React + TypeScript** | Component-driven UI with real-time cost updates |

| **Firebase** | Real-time database for team sync & meeting data storage |
| **Vercel** | Instant deployment, live within hours |
| **Calendar API** | Direct calendar import to audit existing meetings |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/MansiSuryawanshi/MeetingWorthIt.git

# Move into the directory
cd MeetingWorthIt

# Install dependencies
npm install

# Add your environment variables
cp .env.example .env
# → Add your Claude API key and Firebase config

# Start the dev server
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root with the following:

```env
VITE_ANTHROPIC_API_KEY=your_claude_api_key_here
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_APP_ID=your_app_id
```

## 💡 Key Learnings

- **The best ideas come from problems you personally feel** — we've all wasted time in pointless meetings
- **Framing beats features.** *"This meeting costs $24K a year"* hits harder than *"this meeting is inefficient"*
- **Tight deadlines force ruthless prioritization** — we shipped only what actually mattered
- **Claude API shines on context-heavy tasks** — combining role + salary + agenda unlocked surprisingly nuanced outputs
- **Real-time feedback changes behavior instantly** — watching the cost number climb as you add people just *works*
